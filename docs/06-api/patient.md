# 就诊人管理 — API 契约

## 对外接口契约

### 接口 1：创建就诊人
- **Method**：POST
- **Path**：`/api/v1/patients`
- **鉴权**：管理端使用 JWT；H5 端使用 `X-Visitor-Phone` Header（访客手机号）
- **输入**：
  | 参数 | 类型 | 必填 | 说明 |
  |------|------|------|------|
  | name | string | 是 | 姓名，2-20 字符 |
  | id_card | string | 是 | 身份证号，18 位 |
  | phone | string | 是 | 手机号，11 位 |
  | gender | string | 否 | 性别：male/female/unknown |
  | age | int | 否 | 年龄，0-150 |
  | address | string | 否 | 住址，≤200 字符 |
- **输出**：
  ```json
  {
    "code": 200,
    "data": {
      "id": 1,
      "name": "张三",
      "id_card": "110101********1234",
      "phone": "138****8888",
      "gender": "male",
      "age": 32,
      "address": "北京市朝阳区"
    },
    "message": "ok"
  }
  ```
- **错误场景**：
  | 错误码 | 场景 |
  |--------|------|
  | 400 | 参数格式错误（身份证/手机号格式不符） |
  | 409 | 身份证号已存在 |
  | 429 | 创建频率超限 |

### 接口 2：查询就诊人列表
- **Method**：GET
- **Path**：`/api/v1/patients`
- **鉴权**：JWT
- **输入**：
  | 参数 | 类型 | 必填 | 说明 |
  |------|------|------|------|
  | keyword | string | 否 | 按姓名/手机号/身份证号搜索（管理端） |
  | page | int | 否 | 页码，默认 1 |
  | page_size | int | 否 | 每页条数，默认 10，最大 100 |
- **权限**：
  - 移动端：仅返回当前用户关联的就诊人
  - 管理端：返回全部就诊人（支持 keyword 搜索）
- **输出**：
  ```json
  {
    "code": 200,
    "data": {
      "total": 50,
      "list": [
        {
          "id": 1,
          "name": "张三",
          "gender": "male",
          "age": 32,
          "phone": "138****8888"
        }
      ]
    },
    "message": "ok"
  }
  ```

### 接口 3：查询就诊人详情
- **Method**：GET
- **Path**：`/api/v1/patients/:id`
- **鉴权**：JWT
- **权限**：移动端仅限本人关联的就诊人；管理端可查看任意
- **输出**：
  ```json
  {
    "code": 200,
    "data": {
      "id": 1,
      "name": "张三",
      "id_card": "110101********1234",
      "phone": "138****8888",
      "gender": "male",
      "age": 32,
      "address": "北京市朝阳区",
      "created_at": "2026-04-29T10:00:00Z",
      "updated_at": "2026-04-29T10:00:00Z"
    },
    "message": "ok"
  }
  ```
- **错误场景**：
  | 错误码 | 场景 |
  |--------|------|
  | 403 | 无权查看该就诊人 |
  | 404 | 就诊人不存在 |

### 接口 4：更新就诊人
- **Method**：PUT
- **Path**：`/api/v1/patients/:id`
- **鉴权**：JWT
- **输入**：同「创建就诊人」，全部字段可选
- **权限**：移动端仅限本人关联的就诊人；管理端可更新任意
- **错误场景**：
  | 错误码 | 场景 |
  |--------|------|
  | 403 | 无权更新 |
  | 404 | 就诊人不存在 |

### 接口 5：删除就诊人
- **Method**：DELETE
- **Path**：`/api/v1/patients/:id`
- **鉴权**：JWT
- **权限**：移动端仅限本人关联的就诊人；管理端可删除任意
- **错误场景**：
  | 错误码 | 场景 |
  |--------|------|
  | 400 | 该就诊人存在未完成的挂号订单，禁止删除。未完成状态包括：`confirmed`（待就诊） |
  | 403 | 无权删除 |
  | 404 | 就诊人不存在 |
