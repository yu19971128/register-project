# layout — 管理端全局布局框架（PC Web）

> **范围声明**：本文档仅描述 **PC 管理端** 的全局布局框架。H5 移动端页面采用独立布局（Header 44px + 底部固定按钮），各 H5 页面设计详见 `docs/04-frontend/features/*.md` 中的对应文档。
>
> 本页面基于全局布局框架 `docs/04-frontend/layout.md` 渲染，不独立设计导航菜单、全局路由、面包屑。

## 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  [Logo]  当天挂号管理系统          [🔔通知] [👤管理员] [退出]  │  ← Header (56px)
├──────────┬──────────────────────────────────────────────────┤
│          │  [面包屑：首页 / 就诊人管理]                        │  ← Breadcrumb (40px)
│  导航菜单  │                                                   │
│  ─────────┤  ┌─────────────────────────────────────────────┐  │
│  📊 首页   │  │                                             │  │
│  👤 就诊人 │  │          主内容区（页面具体内容）              │  │
│  📝 挂号订单│  │                                             │  │
│  ⚙️ 号源管理│  │                                             │  │
│  📈 数据统计│  │                                             │  │
│          │  └─────────────────────────────────────────────┘  │
│ (200px)  │                                                   │
│          │  [© 2026 当天挂号系统 | 帮助中心 | 意见反馈]       │  ← Footer (40px)
└──────────┴──────────────────────────────────────────────────┘
```

## 布局组件清单

| 组件名 | 路径 | 职责 |
|--------|------|------|
| BasicLayout | layouts/BasicLayout.tsx | 全局布局：Header + Sider + Content + Footer |

| GlobalHeader | components/GlobalHeader.tsx | 顶部导航：Logo、通知、用户信息 |

## 布局规格

| 元素 | 尺寸 | 背景色 | 说明 |
|------|------|--------|------|
| Header | 高度 56px | #fff | 底部阴影 0 1px 4px rgba(0,0,0,0.1) |
| Sider | 宽度 200px（收起 80px） | #001529 | antd 暗色菜单主题 |
| Content | 自适应 | #f0f2f5 | 内边距 24px |
| Footer | 高度 40px | #f0f2f5 | 文字居中，字号 12px |

## 交互状态

| 元素 | 状态 | 样式 |
|------|------|------|
| 菜单项 | hover | 背景色 rgba(255,255,255,0.1) |
| 菜单项 | active/selected | 背景色 #1890ff |
| 菜单项 | 收起状态 | 仅显示图标，tooltip 显示文字 |

## 面包屑规则

- 自动根据当前路由生成面包屑
- 首页固定为第一项
- 末级不可点击
- 最多显示 3 级，超出折叠

## 权限控制点

- 管理端所有页面需登录（JWT 校验）
- 未登录重定向至 /login
- Token 过期自动跳转登录页并提示
- 当前阶段无细粒度 RBAC，仅区分「已登录/未登录」
