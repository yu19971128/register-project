# 就诊人管理 — 前端设计

## 用途
提供就诊人档案的前端管理界面，覆盖移动端（患者自助管理）和管理端（管理员全局管理）。

---

## 页面清单

| 页面 | 端 | 路由 |
|------|-----|------|
| 就诊人列表 | H5 | /h5/patients |
| 添加/编辑就诊人 | H5 | /h5/patients/edit |
| 就诊人管理列表 | PC | /patients |
| 就诊人详情 | PC | /patients/:id |

---

│  │ [编辑] [删除]        │   │  ← 就诊人卡片
│  └─────────────────────┘   │
│  ┌─────────────────────┐   │
│  │ 👤 李四（家属）      │   │
│  │ 女 · 58岁 · 139****6666│  │
│  │ [编辑] [删除]        │   │
│  └─────────────────────┘   │
│                             │
│         ···                 │
│                             │

### 说明
- 顶部返回箭头 + 标题

- 卡片右侧/底部提供编辑、删除操作
- 底部固定「添加就诊人」大按钮

| 表格行 | 背景 #fafafa | — | — | — |
| 输入框 | 边框 #4096ff | 边框 #1677ff | 背景 #f5f5f5 | — |

## 响应式断点要求

| 断点 | 布局变化 |
|------|----------|
| ≥1440px | 管理端内容区最大宽度 1200px，居中 |
| ≥1024px | 管理端标准侧边栏展开 |
| ≥768px | 管理端侧边栏可收起；H5 保持单列 |
| <768px | H5 单列全宽；管理端侧边栏隐藏，汉堡菜单触发 |
