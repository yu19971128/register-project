# 号源管理 — 前端设计

## 用途
提供管理端当天号源的配置界面，支持号源的增删改查及余量监控。

---

## 页面清单

| 页面 | 端 | 路由 |
|------|-----|------|
| 号源管理列表 | PC | /schedules |
| 添加/编辑号源 | PC | /schedules/edit |

---

## 页面 1：号源管理列表（PC 管理端）

**本页面基于全局布局框架 `docs/04-frontend/layout.md` 渲染。**

### 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  [Logo] ... [👤管理员]                                        │  ← GlobalHeader
├──────────┬──────────────────────────────────────────────────┤
│  导航菜单  │  首页 / 号源管理                                  │  ← Breadcrumb
│  ─────────┤                                                   │
│  ...     │  ┌─────────────────────────────────────────────┐  │
│          │  │ 号源管理                       [+ 新增]       │  │  ← PageHeader
│          │  ├─────────────────────────────────────────────┤  │
│          │  │ 📅 日期：[2026-04-29 ▼]    [🔍 查询]         │  │  ← FilterBar
│          │  ├─────────────────────────────────────────────┤  │
│          │  │ 科室    │ 医生   │ 时间段    │ 总号数 │ 余量 │ 状态 │ 操作 │  │  ← Table
│          │  │ ────────┼────────┼───────────┼────────┼──────┼──────┼────── │  │
│          │  │ 内科    │ 王医生 │ 09:00-10:00│ 20     │ 15   │ 可约 │ 编辑 │  │
│          │  │ 外科    │ 李医生 │ 10:00-11:00│ 15     │ 0    │ 已满 │ 编辑 │  │
│          │  └─────────────────────────────────────────────┘  │
│          │  [1][2][3]...  共 20 条                            │  ← Pagination
│          └──────────────────────────────────────────────────┘
```

### 说明
- 默认展示当天号源，日期选择器可切换查看其他日期
- 表格展示科室、医生、时间段、总号数、余量、状态（可约/已满/已停诊）
- 操作列提供编辑、删除（有余量被预约时禁止删除）
