# 当天挂号 — 前端设计

## 用途
提供移动端（H5）当天挂号的完整前端流程，包括号源浏览、挂号确认和挂号凭证展示。

---

## 页面清单

| 页面 | 端 | 路由 |
|------|-----|------|
| 号源浏览 | H5 | /h5/register |
| 挂号确认 | H5 | /h5/register/confirm |
| 挂号凭证 | H5 | /h5/register/ticket |

---

## 页面 1：号源浏览（H5）

### 布局结构

```
┌─────────────────────────────┐
│  ← 当天挂号                │  ← Header (44px)
├─────────────────────────────┤
│  📅 2026-04-29 (今天)       │  ← 日期栏（仅展示当天，不可切换）
├─────────────────────────────┤
│  [全部] [内科] [外科] [儿科] │  ← 科室筛选标签
├─────────────────────────────┤
│  ┌─────────────────────┐   │
│  │ 🩺 内科              │   │
│  │ 王医生 · 09:00-10:00 │   │
│  │ 余号 15/20          │   │  ← 号源卡片
│  │ [立即挂号]           │   │
│  └─────────────────────┘   │
│  ┌─────────────────────┐   │
│  │ 🩺 外科              │   │
│  │ 李医生 · 10:00-11:00 │   │
│  │ 余号 0/15  [已满]    │   │  ← 已满状态不可点击
│  └─────────────────────┘   │
│         ···                 │
└─────────────────────────────┘
```

### 说明
- 顶部返回箭头 + 标题
- 日期栏展示当天日期，不可切换（系统聚焦当天挂号）
- 科室标签横向滚动筛选
- 号源卡片展示科室、医生、时间段、余量/总数
- 余量 > 0 显示「立即挂号」按钮；余量 = 0 显示「已满」且不可点击

## 颜色与样式约束

同 `docs/04-frontend/features/patient.md`，补充：

| 元素 | 颜色值 | 说明 |
|------|--------|------|
| 成功提示 | #52c41a | 绿色 |

| 号源卡片-已满 | #f5f5f5 | 灰色背景，文字置灰 |

## 交互状态说明

同 `docs/04-frontend/features/patient.md`，补充：

| 元素 | 状态 | 样式 |
|------|------|------|
| 号源卡片 | hover/press | 阴影加深，缩放 0.98 |
| 科室标签 | active | 背景 #1677ff，文字白色 |
| 就诊人单选 | selected | 边框 #1677ff，左侧蓝色竖条 |

## 响应式断点要求

H5 页面仅面向移动端，固定单列布局，最大宽度 430px 居中。
