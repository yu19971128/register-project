# 模块：就诊人管理

## 职责
- 提供就诊人档案的全生命周期管理：创建、查询、更新、删除。
- **移动端（H5）**：患者可添加、编辑、删除自己的就诊人（如本人、家属），填写姓名、身份证号、手机号、性别、年龄、住址等信息。
- **管理端（PC）**：管理员可查看所有就诊人档案列表，支持按姓名/手机号/身份证号搜索，可查看详情，可编辑和删除档案。
- 敏感信息（身份证号、手机号）加密存储，仅脱敏展示。

## 边界
- **不做挂号逻辑**：就诊人与挂号的关联关系由「当天挂号」模块维护，本模块只提供就诊人基础数据。
- **不做号源管理**：号源配置、余量扣减属于「号源管理」模块。
- **不做订单状态流转**：挂号订单的退号/改号状态由「挂号订单管理」模块处理。
- **不做实名认证**：仅做数据格式校验（身份证格式、手机号格式），不对接公安/运营商实名接口。

## 依赖模块
- 无（基础模块，被其他模块依赖）
